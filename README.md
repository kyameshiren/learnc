# learnc
## Chapter 1
- helloworld : Hello World!
- temp : Makes temperature conversion table
- char : Converts multiple blanks into single blank
- copy : Copies input to output
- count : Counts \n \d \t \c in file
- spec : Prints ASCII value of character
- repl : Replaces tabs with \t
- word : Counts word in file
- wpl : Prints input one word per line
